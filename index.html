Generating Locators Made Easy

The Painful Locator Hunt

Projects weren't built with automation in mind

  • Manual element inspection using browser dev tools
  • Reliance on console logs, Chrome extensions, and trial-and-error methods
  • More time spent finding locators than writing actual tests
Hours of clicking around

Locator Generation with Augment

AI-powered automation that understands your needs

  • Simply describe what you need and Augment generates all locators
  • AI can write complete test scripts (though the team prefers crafting them manually)
  • Result: Less time clicking around, more time building value
Minutes to get started
Focus on test logic, not element hunting
AI understands context and generates reliable selectors
Faster test development and maintenance
